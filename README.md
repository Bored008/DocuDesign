# DocDesign

Convert static document images into fully **editable documents**.

DocDesign is a web-based document editor that allows users to convert a static document image (such as a resume or project front page) into an editable layout. Instead of recreating the document from scratch, users can directly edit the text, adjust formatting, and export the final result.

The tool is designed to be simple, fast, and usable locally without storing user data on a server.

## Features

### Editable Canvas

Users can edit document content directly inside the editor interface.

### Export Options

Documents can be exported as:

* Image
* PDF
* Word document

### Local Storage Support

All document edits can be stored locally on the user's device. No server storage required.

### Free to Use

DocDesign can run locally on your system or be deployed on a live server.

### Quick Editing

Instead of recreating documents from scratch, users can simply edit the existing layout.

